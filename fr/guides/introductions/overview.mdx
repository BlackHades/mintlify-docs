Nous fournissons l'infrastructure nécessaire pour intégrer les moyens de paiement alternatifs et locaux en Afrique, notamment le mobile money, les virements bancaires et les cartes, via une seule intégration multi-PSP. ## Guide de démarrage rapide

Mettez-vous en route en trois étapes simples :

<Steps>
  <Step title="Créer votre compte sandbox">
    Inscrivez-vous sur [sandbox-app.payfonte.com](https://sandbox-app.payfonte.com) pour accéder à votre environnement de test.
  </Step>

  <Step title="Récupérer vos identifiants API">
    Ouvrez `Settings -> API Keys/Webhooks` dans votre dashboard pour récupérer `client-id` et `client-secret`.
  </Step>

  <Step title="Faire votre premier appel API">
    Utilisez l'URL de base sandbox pour tester votre intégration :

    ```bash
    curl --location 'https://sandbox-api.payfonte.com/payments/v1/checkouts' \
      --header 'client-id: <your-client-id>' \
      --header 'client-secret: <your-client-secret>' \
      --header 'Content-Type: application/json' \
      --data '{
        "reference": "test-001",
        "amount": 1000,
        "currency": "NGN",
        "country": "NG",
        "redirectURL": "https://yoursite.com/callback",
        "user": {
          "email": "customer@example.com",
          "phoneNumber": "08012345678"
        }
      }'
    ```
  </Step>
</Steps>

<Note>
  Prêt pour la production ? Terminez le processus KYB via notre [formulaire d'onboarding](https://form.jotform.com/240038360690048) puis passez en live sous 1 à 5 jours ouvrés.
</Note>

## Moyens de paiement alternatifs pris en charge

Payfonte prend en charge de nombreux moyens de paiement locaux en Afrique via son agrégateur mobile money :

| Catégorie | Providers |
|----------|-----------|
| Mobile Money API | MTN MoMo, Orange Money, Airtel Money, M-Pesa, Wave, Moov, Togocel, Halopesa |
| Virements bancaires | Virements bancaires directs, Nigeria, Kenya, Afrique du Sud |
| Cartes | Visa, Mastercard via des gateways partenaires |
| Portefeuilles | Opay, PalmPay, OPay Wallet |

Nos intégrations mobile money couvrent notamment :

- MTN MoMo dans plus de 10 pays
- M-Pesa pour le Kenya, la Tanzanie et d'autres marchés d'Afrique de l'Est
- Airtel Money pour une couverture panafricaine
- Wave pour l'Afrique de l'Ouest
- Orange Money, Moov et d'autres providers régionaux
