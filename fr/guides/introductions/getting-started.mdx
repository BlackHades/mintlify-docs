## Qu'est-ce que Payfonte ?

Payfonte est une plateforme unifiée d'orchestration de paiement qui aide les entreprises à collecter et à décaisser via plusieurs providers et marchés depuis un seul dashboard et une seule API. Nous simplifions la gestion des passerelles, le routage, le rapprochement, le reporting et l'intégration afin que votre équipe se concentre sur la croissance plutôt que sur la complexité de l'infrastructure de paiement.

## Guide de démarrage rapide

<Steps>
  <Step title="Créer votre compte sandbox">
    Inscrivez-vous sur [sandbox-app.payfonte.com](https://sandbox-app.payfonte.com) pour accéder à votre environnement de test.
  </Step>
  <Step title="Récupérer vos identifiants API">
    Ouvrez **Settings -> Security ->  API Keys and Webhooks** dans votre dashboard pour récupérer `client-id` et `client-secret`.
  </Step>
  <Step title="Faire votre premier appel API">
    Utilisez l'URL de base sandbox pour tester votre intégration :

    ```bash
    curl --location 'https://sandbox-api.payfonte.com/payments/v1/checkouts' \
      --header 'client-id: <your-client-id>' \
      --header 'client-secret: <your-client-secret>' \
      --header 'Content-Type: application/json' \
      --data '{
        "reference": "test-001",
        "amount": 1000,
        "currency": "NGN",
        "country": "NG"
      }'
    ```

  </Step>
</Steps>

<Check>
  **Prêt pour la production ?** Terminez le processus KYB via notre [formulaire

  1 à 5 jours ouvrés.
