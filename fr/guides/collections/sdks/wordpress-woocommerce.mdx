# URL du plugin

Le plugin n'est pas encore publié sur la marketplace, mais vous pouvez télécharger l'archive ZIP ici : https://payfonte.s3.amazonaws.com/woo-payfonte.zip

# Installation ou mise à jour du plugin Payfonte WooCommerce

Suivez ces étapes pour installer le plugin WordPress depuis l'URL fournie.

**Étape 1 : Télécharger le plugin**

Téléchargez d'abord le plugin à l'adresse suivante : https://payfonte.s3.amazonaws.com/woo-payfonte.zip

**Étape 2 : Connectez-vous à votre tableau de bord WordPress**

Connectez-vous ensuite à votre interface WordPress, généralement accessible via `votresite.com/wp-admin`.

**Étape 3 : Ouvrir la section Plugins**

![image](https://files.readme.io/d9fbf02-image.png)

Depuis le dashboard, ouvrez la section "Plugins" dans le menu de gauche.

**Étape 4 : Ajouter un nouveau plugin**

Cliquez sur le bouton "Add New" en haut de la page "Plugins".

**Étape 5 : Téléverser le plugin**

Sur la page "Add Plugins", cliquez sur "Upload Plugin" en haut de la page.

**Étape 6 : Choisir le fichier ZIP**

Cliquez sur "Choose File" dans la section d'upload. Naviguez jusqu'au fichier `woo-payfonte.zip` que vous avez enregistré et sélectionnez-le.

**Étape 7 : Installer le plugin**

Cliquez sur "Install Now" pour lancer l'installation. WordPress décompressera puis installera automatiquement le plugin.

**Étape 8 : Activer le plugin**

Une fois l'installation terminée, cliquez sur "Activate Plugin" pour commencer à utiliser le plugin.

L'installation est terminée.

# Configuration du plugin Payfonte WooCommerce

Après l'installation ou une mise à jour, configurez le plugin avec vos identifiants.

1. Cliquez sur le lien `Settings` sous `Payfonte Woocommerce` dans la section des plugins, ou ouvrez `Woocoomerce -> Settings -> Payments -> Payfonte Woocommerce`.

   ![image](https://files.readme.io/1a5de8e-image.png)

2. Une fois sur l'écran de configuration, vous verrez les champs correspondants. Vous pouvez récupérer vos clés API en suivant les étapes de [Autorisation](/fr/guides/introductions/authorization).

3. ![image](https://files.readme.io/e39a392-image.png)

4. Si `test mode` est activé, récupérez vos identifiants sandbox sur https://sandbox.payfonte.com

5. Si `test mode` est désactivé, récupérez vos identifiants live sur https://payfonte.com
