</Info>

## At a Glance

<CardGroup cols={2}>
  <Card title="12 Markets" icon="globe-africa">
    Provider coverage is currently published for 12 countries across West, East, and Central Africa.
  </Card>
  <Card title="39 Provider Slugs" icon="fingerprint">
    Send the exact `Slug` value in your API payloads when specifying the `provider` field.
  </Card>
  <Card title="35 Dual-Flow Slugs" icon="right-left">
    Most provider slugs support both `Collection` and `Disbursement`, with a few market-specific exceptions.
  </Card>
  <Card title="1 Fetch Required" icon="sliders">
    `bank-transfer-nigeria` is the only listed provider that currently requires fetch-properties support.
  </Card>
</CardGroup>

<Warning>
  Payfonte does not support decimal amounts. Send integer amounts only in minor units. For example, `100.50 NGN` becomes `10050`. See [Amount Specification](/guides/introductions/amount-specification).
</Warning>

<Note>
  A `—` in the min/max columns means no limit is currently published on this page. Actual transaction limits are typically set by the provider or MNO and may vary based on the end user's account level, so do not treat `—` as a guarantee that no restriction exists.
</Note>

## Column Guide

<AccordionGroup>
  <Accordion title="Provider" icon="building" defaultOpen>
    Human-readable provider name such as MTN MoMo, Orange, Wave, Bank Transfer, or M-Pesa.
  </Accordion>
  <Accordion title="Slug" icon="tag">
    Machine-readable provider identifier used in API calls, for example `mtn-momo-ghana`.
  </Accordion>
  <Accordion title="Currency" icon="coins">
    The transaction currency expected for that country and provider row.
  </Accordion>
  <Accordion title="Type" icon="right-left">
    - `Collection`: collection or charge flow
    - `Disbursement`: payout or transfer flow
  </Accordion>
  <Accordion title="Fetch Properties" icon="sliders">
    `Yes` means you may need to call a provider-properties endpoint before initiating a request.
  </Accordion>
  <Accordion title="Min. Amount / Max. Amount" icon="hashtag">
    Published transaction thresholds where currently defined. Amount values follow the currency minor-unit rules in [Amount Specification](/guides/introductions/amount-specification).
  </Accordion>
</AccordionGroup>

<Info>
  Provider availability can change by market, compliance policy, or partner uptime. Confirm your current dashboard setup before going live.
</Info>
