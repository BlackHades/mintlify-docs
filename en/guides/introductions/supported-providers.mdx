<Warning>
  Payfonte does not support decimal amounts. Send integer amounts only in minor units. For example, `100.50 NGN` becomes `10050`. See [Amount Specification](/en/guides/introductions/amount-specification).
</Warning>

<Note>
  A `—` in any flow-limit column means no limit is currently published on this page. Actual transaction limits are typically set by the provider or MNO and may vary based on the end user's account level, so do not treat `—` as a guarantee that no restriction exists.
</Note>

## Column Guide

<AccordionGroup>
  <Accordion title="Provider" icon="building" defaultOpen>
    Human-readable provider name such as MTN MoMo, Orange, Wave, Bank Transfer, or M-Pesa.
  </Accordion>
  <Accordion title="Slug" icon="tag">
    Machine-readable provider identifier used in API calls, for example `mtn-momo-ghana`.
  </Accordion>
  <Accordion title="Currency" icon="coins">
    The transaction currency expected for that country and provider row.
  </Accordion>
  <Accordion title="Supported Flows" icon="right-left">
    - `Collection`: collection or charge flow only
    - `Disbursement`: payout or transfer flow only
    - `Collection, Disbursement`: the same slug supports both flows

    Every provider row currently listed on this page uses `Collection, Disbursement`.
  </Accordion>
  <Accordion title="Collection Processing Type" icon="route">
    Indicates whether collections use direct processing such as `USSD/STK_PUSH`, redirect, or both.
  </Accordion>
  <Accordion title="Fetch Properties" icon="sliders">
    `Yes` means you may need to call a provider-properties endpoint before initiating a request.
  </Accordion>
  <Accordion title="Flow Limits" icon="hashtag">
    Collection and disbursement limits are shown in separate columns so each provider slug only appears once. Amount values follow the currency minor-unit rules in [Amount Specification](/en/guides/introductions/amount-specification).
  </Accordion>
</AccordionGroup>

<Info>
  Provider availability can change by market, compliance policy, or partner uptime. Confirm your current dashboard setup before going live.
</Info>
